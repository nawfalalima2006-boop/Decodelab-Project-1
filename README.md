DecodeLabs Project 1 – Advanced EDA & Feature Engineering

About the project 
This project was completed as part of my DecodeLabs Data Science Internship.The objective of this project is to perform Exploratory Data Analysis (EDA), data cleaning, preprocessing, and feature engineering to transform raw data into a machine-learning-ready dataset

Objectives:
   •Perform Exploratory Data Analysis (EDA)
   •Understand dataset structure and patterns
   •Handle missing values and duplicate records
   •Detect and handle outliers
   •Apply data preprocessing techniques
   •Create meaningful new features
   •Prepare data for machine learning models
